小优AIGC · cinematic-director — 安装说明
====================================================

cinematic-director（视觉导演）：一句话场景 → 电影摄影级生图 Prompt。
支持 Claude Code 和 Codex CLI（同一个文件夹，两边通用）。


一、安装（拷贝文件夹即完成）
----------------------------------------------------

【Claude Code 用户】
把 cinematic-director 文件夹拷贝到：

    个人级（推荐，所有项目都能用）：
    Windows:  C:\Users\你的用户名\.claude\skills\
    Mac:      ~/.claude/skills/

    或项目级（只在某个项目里用）：
    你的项目\.claude\skills\

【Codex CLI 用户】
把 skill 文件夹拷贝到：

    Windows:  C:\Users\你的用户名\.codex\skills\
    Mac:      ~/.codex/skills/

拷完开一个新会话就能用了。


二、使用方法
----------------------------------------------------

方式1（自动触发）：直接说需求，agent 会自动调用 skill：

    帮我写一个生图提示词：一位23岁清冷女侠站在月下竹林中，低头看剑，中景人像

方式2（手动触发）：
    Claude Code 里输入： /cinematic-director 你的场景描述
    Codex 里输入 /skills 查看，或用 $ 提及 skill

它会先开一场虚拟摄影会议（导演、摄影指导、美术指导逐个发言），
再完成七大模块决策，最后输出 FINAL PROMPT + NEGATIVE PROMPT。

生成结果不满意时，可以只让它重做某一个模块，例如：

    Visual Hook 不好，重新设计

换场景保留风格：

    同样的风格，场景换成雪山之巅


三、不用 Claude Code / Codex？
----------------------------------------------------

用"单文件版skill.md"：把它的全部内容作为系统提示词（System Prompt）
导入 chatGPT、deepseek、豆包 等平台，然后输入：

    Scene: 你的场景
    Style: 风格（可选）
    View: 视角（可选）


四、提示词用在哪些生图模型？
----------------------------------------------------

即梦 / 豆包 / Midjourney / Flux / SDXL / GPT Image / Gemini 都可以。
- 支持负向提示词的模型：正向、负向两段分别粘贴
- 不支持负向的模型（即梦、GPT Image 等）：skill 会自动把"避免项"
  融入正向结尾，直接用正向即可
- Midjourney：负向部分用 --no 参数

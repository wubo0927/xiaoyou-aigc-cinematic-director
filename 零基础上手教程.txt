零基础上手教程 —— 给你的 AI 雇一个摄影剧组
======================================================

这份教程写给完全没碰过命令行的朋友。
跟着走，从装软件到出第一张图，大约 15 分钟。


〇、先搞懂：你在装的是什么？
------------------------------------------------------

普通的 AI 生图，是雇了一个画师：你描述，它画，画成什么样看运气。

你现在要装的，是一整个剧组：

  · Claude Code（或 Codex）是"剧组的办公室"——一个能自己干活的 AI 助手（也叫 agent）
  · cinematic-director 文件夹是"剧组本身"——导演、摄影指导、美术指导，
    加上他们的全套工作手册（人物怎么设计、镜头放哪、什么坚决不拍）

装好之后，你只需要说一句话，剧组会自己开会、翻手册、
最后把一份电影级的生图提示词交到你手里。

注意：办公室需要付费账号（见下），如果暂时不想花钱，
可以先用包里的"单文件版skill.md"——把它的内容作为系统提示词
粘贴进 chatGPT、deepseek、豆包等任意对话软件，也能用（效果是简化版）。


一、准备工作（二选一）
------------------------------------------------------

【选项 A：Claude Code】（本教程主推，文档最全）
  需要：Claude Pro / Max 订阅（在 claude.ai 注册并订阅）

【选项 B：OpenAI Codex】
  需要：ChatGPT Plus 及以上订阅


二、安装"办公室"
------------------------------------------------------

【装 Claude Code】

Windows 用户：
  1. 按键盘 Win 键，输入 PowerShell，回车打开蓝色窗口
  2. 把下面这行整个复制进去，回车：

     irm https://claude.ai/install.ps1 | iex

  3. 等它跑完。然后输入 claude --version 回车，
     能看到版本号就是装好了
  4. 输入 claude 回车，会弹出浏览器让你登录 Claude 账号，
     登录一次以后就不用再登了

Mac 用户：
  1. 打开"终端"（启动台里搜 Terminal）
  2. 复制粘贴这行，回车：

     curl -fsSL https://claude.ai/install.sh | bash

  3. 之后同上：claude --version 验证，claude 登录

【装 Codex（选项 B）】

Mac：终端里运行  curl -fsSL https://chatgpt.com/codex/install.sh | sh
Windows：装法请以官方文档为准（developers.openai.com/codex/cli），
一般可通过 npm 安装。首次运行 codex 后选 "Sign in with ChatGPT" 登录。


三、把"剧组"搬进办公室（三步，这是全部）
------------------------------------------------------

  第 1 步：解压你收到的压缩包

  第 2 步：把整个 cinematic-director 文件夹，拷贝到下面的位置

    Claude Code 用户：
      Windows:  C:\Users\你的用户名\.claude\skills\
      Mac:      ~/.claude/skills/

    Codex 用户：
      Windows:  C:\Users\你的用户名\.codex\skills\
      Mac:      ~/.codex/skills/

    （skills 文件夹如果不存在，自己新建一个就行）

  第 3 步：重新打开 Claude Code（或 Codex），说出你想要的画面

  对，没有第四步。


四、第一次使用
------------------------------------------------------

随便找个文件夹，打开 PowerShell / 终端，输入 claude 回车，然后直接说：

  帮我写一个生图提示词：大雪初停的白桦林，
  一位红衣女子牵着白马走在小路上，望向林深处

接下来你会看到剧组开工的全过程：

  1. 先开一场会（Creative Meeting）——导演说这幕拍什么，
     摄影指导定机位，美术指导定颜色
  2. 它自己去翻手册——读取模板和模块文件
  3. 逐项完成七个决策模块
  4. 最后交给你两段东西：

     FINAL PROMPT（正向提示词）
     NEGATIVE PROMPT（负向提示词）

把这两段分别粘进你的生图软件（即梦 / Midjourney / SD 等）就行。

  · 即梦、豆包等不支持负向的：只贴 FINAL PROMPT 即可，
    它已经把"要避免什么"写在正文结尾了
  · Midjourney：负向内容用 --no 参数带上


五、进阶玩法（剧组的正确使唤方式）
------------------------------------------------------

不满意，别推倒重来，指名道姓地改：

  "Visual Hook 不好，重新设计"        → 只重做视觉钩子这一个模块
  "同样的风格，场景换成雨夜的老街"     → 摄影语言不动，只换场景
  "生成的图太像游戏CG了"              → 它会自己诊断问题出在哪个模块


六、常见问题
------------------------------------------------------

Q：说了场景它没反应/没按剧组流程走？
A：直接输入 /cinematic-director 加你的场景描述，手动召唤剧组。

Q：Windows 找不到 .claude 文件夹？
A：打开文件资源管理器，直接在地址栏输入
   C:\Users\你的用户名\.claude  回车。还是没有的话，
   先运行一次 claude 再看；skills 子文件夹可以自己新建。

Q：完全免费能玩吗？
A：办公室（Claude Code / Codex）需要订阅。不想付费就用
   "单文件版skill.md" 粘进免费的对话软件，先体验简化版。

Q：我用的是 Hermes Agent，能装吗？
A：能，同一个标准。拷到 ~/.hermes/skills/cinematic-director 即可。
   提醒：Hermes 里配的底层模型越强，剧组发挥越好；
   本地小模型可能会偷懒跳步。

Q：听说还有编剧和分镜 skill？
A：有——它们是这条流水线的前两级：编剧把一句话想法变成完整剧本，
   分镜把剧本拆成逐镜头脚本。这两个不在本包里，
   介绍、实测效果图和获取方式见 GitHub 仓库首页
   （最顶上的「交流与合作」）。

Q：以后怎么拿最新版？
A：这个剧组会持续进化，拍得好的经验会写进它自己的手册里。
   在 GitHub 搜索 xiaoyou-aigc-cinematic-director（或直接搜：小优AIGC），
   进入仓库后点绿色 Code 按钮 → Download ZIP，
   解压后按"三步拷贝"覆盖旧文件夹即可。

Q：安装命令跑不通？
A：安装方式可能随版本更新，以官方文档为准：
   Claude Code → code.claude.com/docs
   Codex     → developers.openai.com/codex

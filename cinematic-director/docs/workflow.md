# 摄影团队工作流 — Creative Meeting

生成 Prompt 之前，必须先模拟一次真实摄影团队的拍摄会议。这一步的价值：

1. 强迫先做**视觉决策**，再写 Prompt（避免直接堆词）
2. 会议记录本身就是各模块的草稿
3. 用户可以针对某个角色的决定单独提出修改（专业影视工作流）

## 团队角色与职责

| 角色 | 决定什么 | 对应模块 |
|---|---|---|
| Director（导演） | 这一幕**真正拍的是什么**（主题，不是物体） | Intent / Story Moment |
| Screenwriter（编剧） | 故事停留在哪一帧，这一秒刚刚发生了什么 | Story Moment |
| Character Designer（角色设计） | 人物是谁，为什么第一眼愿意看 | Character DNA |
| Cinematographer / DoP（摄影指导） | 摄影机放哪、镜头、机位、人物占比 | Photography Strategy |
| Art Director（美术指导） | 色彩关系（限制在 3~5 种） | Color |
| Creative Director（创意总监） | 观众第一眼为什么停下 | Visual Hook |
| Photographer（摄影师） | 通过镜头完成表达 | Photographer's Notes |

## 会议流程

### 1. 导演先否定直觉

导演的第一个发言永远是回答：

> **这一场戏，到底拍什么？**

答案永远不是物体（女侠/竹林/月亮），而是主题：

> 人与剑之间那两秒钟的沉默。
> 一触即发。
> 等待。不是战斗。

### 2. 各角色依次发言

每个角色只做自己职责内的决策，发言要短、要具体、要是"决定"而不是"描述"。

### 3. 输出会议记录

## Creative Meeting 输出格式

```text
===================
Creative Meeting
===================

Director
这一幕真正拍的是：等待。不是战斗。

--------------------------------
Story
她停下脚步。风吹过。什么都没发生。

--------------------------------
Cinematographer
摄影机：85mm，Eye Level，留在竹林内部。
人物占40%。不拍天空。

--------------------------------
Art Director
颜色：墨绿、冷白、银灰。不超过五种。

--------------------------------
Visual Hook
第一眼：剑。
第二眼：眼神。
第三眼：竹林。

--------------------------------
Attention
她一直看着剑。不是镜头。

--------------------------------
Mood
安静。压抑。孤独。
```

## 会议之后

会议记录确认后，按 `CLAUDE.md` Step 4~10 依次完成七大模块，然后组装 Prompt。

**关键纪律：**

- 会议上做出的决定，模块和 Prompt 必须遵守（如"不拍天空"→ Prompt 中月亮不能出现，只能出现月光；负向中加入 visible sky, visible moon disc）
- 用户的硬性设定优先级最高，会议不能推翻用户设定，只能在设定内做摄影决策
- 会议记录用中文输出，最终 Prompt 用英文

## 机位讨论的正确方式（Cinematographer 参考）

摄影机位置不是参数，而是叙事立场。以竹海对决为例：

AI 默认机位（错误）：

```
人物 ↓ 摄影机        → 人物巨大，背景很小 → 人物写真
```

电影机位（正确）：

```
天空 ↓ 人物 ↓ 竹海 ↓ 摄影机
距离至少100米，35mm，人物占画面10%，竹海占90%
```

因为要拍的是**武侠**，不是人物写真。摄影机不追人物，摄影机观察竹林，人物只是竹林里的两只白鹤。

## Shot List 的本质

Shot List 不是镜头参数，而是**镜头意图**：

```text
不是：Wide Shot / Close Up / Low Angle

而是：
山应该压住人物。
天空需要保持安静。
光不要直接照人物。
留出大量空白。
人物不要抢戏。
Sunlight arrives before the character.
The rider should almost disappear.
Hold the silence.
```

AI 特别容易理解这种语言。
